## 删除文件
rm 命令可以通过“取消 Pin（unpin）”来从 PinMe 的 IPFS 节点移除内容，并清理关联的 ENS 域名记录。这有助于你管理存储并保持部署记录整洁。

### 命令语法
```bash
pinme rm [hash]
```
### 参数
- hash（可选）：要删除的 IPFS 内容哈希（CID）

### 理解“删除”
#### 删除时会发生什么？
执行 pinme rm 时，通常会做以下事情：

- 取消 Pin：内容会从 PinMe 的 IPFS 节点取消固定
- 域名清理：移除关联的 ENS 域名记录
- 历史更新：上传历史会标记为已删除
- 缓存清理：本地与云端缓存条目会清除

**⚠ 重要限制**
IPFS 是分布式网络。从 PinMe “删除”意味着：

- 从 PinMe 的 IPFS 节点移除内容
- 删除关联的域名记录
- 清理你的上传历史
- [ERROR] 不能保证从整个 IPFS 网络完全消失
- [ERROR] 不能移除其他人已经 Pin 的副本
- 内容一旦进入 IPFS，其他用户可能已固定它，或它已存在于其他节点上。

### 删除操作
1. 交互式删除
2. 按哈希删除
```bash
# 通过 IPFS 哈希删除指定内容
pinme rm bafybeiabc123def456ghi789jkl012mno345pqr678stu

# 使用短哈希（若唯一）
pinme rm bafybeiabc
3. 按 ENS 地址删除
# 通过 ENS 地址删除
pinme rm xxx.pinme.eth.limo

# 或者输入完整 URL
pinme rm https://xxx.pinme.eth.limo/
```
