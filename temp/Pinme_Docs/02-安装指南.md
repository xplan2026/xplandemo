# 安装指南
本指南将带你在本地安装 PinMe。PinMe 以 Node.js 包形式发布，可通过 npm 或 yarn 安装。

## 前置条件
### 系统要求
- 操作系统：Windows、macOS 或 Linux
- Node.js：16.0 或更高版本
- 网络连接：上传到 IPFS 与通过网关访问需要联网

### 安装 Node.js
如果你还没有安装 Node.js，请从官网下载安装：
```bash
https://nodejs.org/
```
建议使用 LTS（长期支持）版本以获得更稳定体验。

### 验证 Node.js 安装
```bash
node --version
# 应输出：v16.0.0 或更高

npm --version
# 应输出：8.0.0 或更高
```


## 安装方式

### 方式 1：使用 npm（推荐）
```bash
npm install -g pinme
```
这会把 PinMe 全局安装到你的系统，使 pinme 命令在任意目录都可用。

### 方式 2：使用 yarn
如果你更偏好使用 yarn：
```bash
yarn global add pinme
```
### 安装验证
安装完成后，可通过以下方式确认 PinMe 已正确安装：
```bash
# 查看版本号
pinme --version
# 查看帮助
pinme help
```
如果能正常输出帮助信息且无报错，则 PinMe 安装成功。

## 故障排查
### 权限不足（Linux/macOS）
如果安装时提示权限错误：
```bash
sudo npm install -g pinme
```
或者使用 Node 版本管理器，如 nvm：
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装并使用最新 Node.js
nvm install node
nvm use node

# 安装 PinMe
npm install -g pinme
```


### 命令找不到（Command Not Found）
如果安装后找不到 pinme 命令：

检查 npm 全局路径：
```bash
npm config get prefix
```
加入 PATH（如有需要）：
```bash
# bash
echo 'export PATH=$(npm config get prefix)/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# zsh
echo 'export PATH=$(npm config get prefix)/bin:$PATH' >> ~/.zshrc
source ~/.zshrc
#重启终端或执行 
source ~/.bashrc（或 ~/.zshrc）
```
### Windows 特有问题
1. Windows PowerShell 执行策略
如果 PowerShell 报执行策略错误：
```bash
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
2. Windows PATH 配置
- 在“系统属性”里打开“环境变量”
- 在用户变量中找到 “Path”
- 添加：%APPDATA%\npm
- 更新 PinMe
```bash
# 更新到最新版本：
npm update -g pinme

#或者使用 yarn：
yarn global upgrade pinme
```
### 卸载 PinMe
如需移除 PinMe：
```bash
npm uninstall -g pinme

#或者使用 yarn：
yarn global remove pinme
```
