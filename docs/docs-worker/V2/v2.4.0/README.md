## 版本特性

### v2.4.0 主要变更

1. **API Key 认证**
   - 新增 API Key 认证机制
   - 保护危险操作端点（`/scan`, `/restart`）

2. **RPC 节点池优化**
   - 支持从环境变量 `BSC_RPC_NODES` 读取节点列表
   - 修复 RPC 节点连续初始化的问题

3. **Worker 重启功能**
   - 新增 `/restart` 端点
   - 支持清除分布式锁和缓存

4. **分布式锁管理**
   - 改进分布式锁机制
   - 支持全局扫描锁和手动扫描锁
